用于将Click配置文件加载到该VNF所在的虚机中

参数
type字符串，指明该VNF在SFC链中的位置，链首链尾或链间

vnfConfig字符串，内容为Click配置文件
instanceIp字符串，NAT设备的外网地址，用于构成Click配置文件中AddressInfo组件中的内容
internServer字符串，NAT设备Click配置文件中AddressIndo组件中内网服务器的IP地址
localIp字符串，配置节点可访问的该VNF所在虚拟机的IP地址
configIp字符串，配置节点的IP地址
返回
字符串，返回RPC请求的结果

为每个VNF所在虚机创建虚拟网卡，并配置网卡MAC和IP，添加流表规则来规定数据包在这些网卡中的传输

参数
type字符串，用于指示该网元在SFC中的位置，分为链首，链尾和链间
localIp字符串，用于指明该VNF所在虚机的IP地址
backIp字符串，SFC链中下一跳VNF所在虚机的IP地址
forwardIp字符串，SFC链中上一条VNF所在虚机的IP地址
vnfType字符串，该VNF所承载的网元功能，如Firewalls、NAT等
instanceIp2字符串，当SFC类型为跨域通信类型时，需要此参数，该IP是一个OpenStack的浮动IP，用于将OpenStack内网流量的IP改写为外网能访问的浮动IP
internServer字符串，用于告知NAT设备内网服务器的IP地址，当内外网访问流量是针对内网服务器时进行相应的流量地址改写和传输
configIp字符串，用于指明配置节点的IP地址，从而完成RPC通信
返回
字符串，返回RPC请求的结果
