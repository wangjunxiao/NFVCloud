RxJS Aggregates Module

The Reactive Extensions for JavaScript has a number of aggregation operators including those you might already know from the Array::extras and the upcoming ES6 standard such as reduce, find and findIndex. This module is used exclusively for aggregation operations used on finite observable sequences. In addition to the aforementioned operators, there are many useful operators such as count, sum, average and determining whether two sequences are equal via the sequenceEqual method. This requires rx.lite.js from the rx-lite NPM module.

Getting Started

There are a number of ways to get started with RxJS.

Installing with NPM

{bash`}
$ npm install rx-lite-aggregates
$ npm install -g rx-lite-aggregates

Using with Node.js and Ringo.js

var Rx = require('rx-lite-aggregates');

In a Browser:

<script src="path/to/rx.lite.js"></script>
<script src="path/to/rx.lite.aggregates.js"></script>
