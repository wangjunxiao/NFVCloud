RegJSGen

Generate RegExps from RegJSParser’s AST.

Installation

npm install --save regjsgen

Usage

``‘js var regjsgen = require('regjsgen’); // With regjsparser var regjsparser = require('regjsparser'); var regex = '^a$'; var ast = regjsparser.parse(regex); // Modify AST // ... // Regenerate RegExp regex = regjsgen.generate(ast);

## See Also
* [RegJSParser](https://github.com/jviereck/regjsparser)
* [RegExp.js](https://github.com/jviereck/regexp.js)
