#define PFR_LENGTH   23

Flash Socket Policy Apache Module.

This module provides a flash socket policy file on the same port that serves HTTP on Apache. This can help simplify setting up a server that supports cross-domain communication with flash.

Quick note about Apache memory handling: Data is allocated from pools and is not manually returned to those pools. The pools are typically considered short-lived and will be cleaned up automatically by Apache.
