aria-role

Elements with ARIA roles must use a valid, non-abstract ARIA role. A reference to role defintions can be found at WAI-ARIA site.

AX_ARIA_01 DPUB-ARIA roles

Rule details

This rule takes one optional object argument of type object:

{
"rules": {
"jsx-a11y/aria-role": [ 2, {
"ignoreNonDOM": true
}],
}
}

For the ignoreNonDOM option, this determines if developer created components are checked.

### Succeed

<div role="button"></div>
<div role={role}></div>
<div></div>
<Foo role={role}></Foo>

Fail

<div role="datepicker"></div>
<div role="range"></div>
<div role=""></div>
<Foo role={role}></Foo>
