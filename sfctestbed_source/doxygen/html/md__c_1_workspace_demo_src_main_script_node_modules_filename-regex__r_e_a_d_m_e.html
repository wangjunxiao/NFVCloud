Regular expression for matching file names, with or without extension.

Install

Install with npm:

$ npm install --save filename-regex

Usage

var regex = require('filename-regex');
'a/b/c/d.min.js'.match(regex());
//=> match[0] = 'd.min.js'
'a/b/c/.dotfile'.match(regex());
//=> match[0] = '.dotfile'
