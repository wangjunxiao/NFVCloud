\begin{quote}
Get the package.\+json of a package from the npm registry \end{quote}


\subsection*{Install}


\begin{DoxyCode}
$ npm install --save package-json
\end{DoxyCode}


\subsection*{Usage}


\begin{DoxyCode}
const packageJson = require('package-json');

packageJson('pageres', 'latest').then(json => \{
    console.log(json);
    //=> \{name: 'pageres', ...\}
\});

// also works with scoped packages
packageJson('@company/package', 'latest').then(json => \{
    console.log(json);
    //=> \{name: 'package', ...\}
\});
\end{DoxyCode}


\subsection*{A\+PI}

\subsubsection*{package\+Json(name, \mbox{[}version\mbox{]})}

You can optionally specify a version (e.\+g. {\ttfamily 1.\+0.\+0}) or {\ttfamily latest}. ~\newline
If you don\textquotesingle{}t specify a version you\textquotesingle{}ll get the \href{http://registry.npmjs.org/pageres/}{\tt main entry} containing all versions.

The version can also be in any format supported by the \href{https://www.npmjs.com/package/semver}{\tt semver} module. For example\+:


\begin{DoxyItemize}
\item {\ttfamily 1} -\/ get the latest {\ttfamily 1.\+x.\+x}
\item {\ttfamily 1.\+2} -\/ get the latest {\ttfamily 1.\+2.\+x}
\item {\ttfamily $^\wedge$1.2.\+3} -\/ get the latest {\ttfamily 1.\+x.\+x} but at least {\ttfamily 1.\+2.\+3}
\item {\ttfamily $\sim$1.2.\+3} -\/ get the latest {\ttfamily 1.\+2.\+x} but at least {\ttfamily 1.\+2.\+3}
\end{DoxyItemize}

\subsection*{Authentication}

Both public and private registries are supported, for both scoped and unscoped packages, as long as the registry uses either bearer tokens or basic authentication.
