\subsection*{A\+PI}

\subsubsection*{xml\+Char\+Classes}

Contains the following regular expressions for matching the different \href{http://www.w3.org/TR/2006/REC-xml-20060816/#CharClasses}{\tt character classes} in the X\+ML spec.


\begin{DoxyItemize}
\item letter
\item base\+Char
\item ideographic
\item combining\+Char
\item digit
\item extender
\end{DoxyItemize}
