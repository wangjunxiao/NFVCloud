\subsection*{Install}

With \href{https://npmjs.org/package/postcss-discard-empty}{\tt npm} do\+:


\begin{DoxyCode}
npm install postcss-discard-empty --save
\end{DoxyCode}


\subsection*{Example}

For more examples see the \href{test.js}{\tt tests}.

\subsubsection*{Input}


\begin{DoxyCode}
@font-face;

\{color:blue\}

h3 \{color:red\}
\end{DoxyCode}


\subsubsection*{Output}


\begin{DoxyCode}
h3 \{color:red\}
\end{DoxyCode}
